## 个人主页（GitHub Pages）

这是一个纯静态个人主页模板（无依赖），适合直接部署到 GitHub Pages。

### 部署到 GitHub Pages（最简单：网页上传）
1. 在 GitHub 新建仓库：
   - **仓库名**建议：`<你的GitHub用户名>.github.io`
   - 例如：`mosery.github.io`
2. 把本目录下文件上传到仓库根目录（包含 `index.html`、`assets/`）。
3. 打开仓库 Settings → Pages：
   - Build and deployment 选择 **Deploy from a branch**
   - Branch 选择 `main` / `root`
4. 等 1-2 分钟访问：`https://<你的GitHub用户名>.github.io`

### 自定义内容
- 编辑 `index.html`：改姓名、简介、项目、联系方式
- 编辑 `assets/style.css`：改配色与排版
