农业物联网监控管理系统

面向设备与基地的实时监控与数据分析：设备状态、历史曲线、预警规则与告警处理。

看实时 联系我
技术栈：Vue2 / Element UI / WebSocket / ECharts
建议展示点
实时推送 告警链路 多图表分析 异常与兜底
如无公网环境：用截图 + 关键交互描述即可。

概览

展示结构越清晰，越容易让面试官快速“代入”。

功能模块

  • 设备监控：在线/离线、指标概览与详情。
  • 历史查询：按设备/时间维度查询，曲线对比。
  • 预警规则：阈值/组合条件配置与启停。
  • 数据分析：多图表联动、维度筛选。

实现要点

  • WebSocket 实时推送与断线重连、心跳/兜底策略（可描述思路）。
  • ECharts 多图表与时间轴交互，支持对比与筛选。
  • Element UI 表格/表单复杂交互与可用性优化。

实时监控（占位）

建议放：设备列表页、设备详情页（实时指标卡 + 曲线）。

页面：设备列表
筛选（基地/设备类型/状态）→ 实时状态点亮 → 点击进入详情
页面：设备详情
实时指标卡（温湿度等）→ 秒级刷新/推送 → 最近告警提示

预警规则（占位）

建议放：规则配置表单、规则列表、告警处理页。

  • 阈值/区间/连续次数条件说明（示例即可）。
  • 告警通知方式、频率控制（若有）。
  • 异常数据与设备离线的处理策略。

数据分析（占位）

建议放：多图表联动、按时间/设备维度对比。

图表：时间序列
区间缩放、tooltip 联动、同屏多指标对比
图表：维度分析
按设备/基地聚合，对比 TopN、异常点标注
